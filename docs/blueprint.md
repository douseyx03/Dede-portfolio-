# **App Name**: Dédé Couro Diop - Ingénieur Électronique

## Core Features:

- En-tête de profil: Afficher une photo professionnelle et les informations de contact de manière visible.
- Sections CV: Présenter les compétences clés, la formation et l'expérience professionnelle dans un format de chronologie clair.
- CV téléchargeable: Permettre le téléchargement du CV au format PDF.

## Style Guidelines:

- Couleur primaire : Bleu foncé (#224563) pour le professionnalisme et la confiance.
- Couleur secondaire : Gris clair (#F0F0F0) pour un arrière-plan propre.
- Accent : Teal (#008080) pour mettre en évidence les informations importantes.
- Utiliser une mise en page propre et moderne avec des sections claires.
- Utiliser des icônes professionnelles pour représenter différentes compétences et expériences.
- Transitions subtiles entre les sections lors du défilement.
